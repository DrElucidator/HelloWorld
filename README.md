# 👋 Hello World Console App

Este repositório contém um simples aplicativo de console em **C#** que imprime mensagens na tela.  
O objetivo deste projeto é demonstrar o funcionamento básico de entrada e saída em aplicações de console.

---

## ✨ Funcionalidades

- **Limpeza da tela:** Uso de `Console.Clear()` para iniciar com a tela limpa.
- **Mensagens no console:** Exibe "Hello, World!" e "Goodbye, World!".
- **Interação com o usuário:** Aguarda o pressionamento da tecla **Enter** antes de encerrar.

---

## 📂 Estrutura do Projeto

  - `Program.cs`: contém toda a lógica do aplicativo.
  - `Console.Clear()`: limpa a tela antes de iniciar.
  - `Console.WriteLine("Hello, World!")`: imprime a mensagem inicial.
  - `Console.ReadLine()`: aguarda o usuário pressionar **Enter**.
  - `Console.WriteLine("Goodbye, World!")`: imprime a mensagem final.

---

## 🚀 Como Executar

1. Abra o terminal na pasta do projeto.
2. Compile o projeto com o comando:
   ```bash
   dotnet build
   ```
3. Execute o aplicativo:
   ```bash
   dotnet run
   ```

---

## 📌 Requisitos

- [.NET SDK 6.0 ou superior](https://dotnet.microsoft.com/download).

---

## 🎯 Objetivos de Aprendizado

- Compreender a estrutura básica de um programa em C#.
- Utilizar comandos de saída (`Console.WriteLine`).
- Trabalhar com entrada de usuário (`Console.ReadLine`).
- Criar um fluxo simples de execução em console.
